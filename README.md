# NullWave

Welcome to NullWave, a highly optimized, fully featured streaming service providing premium music experiences across devices.

## Features
- **High-Quality Audio**: Streams at 320kbps for pristine listening.
- **Smart Recommendations**: Leveraging native recommendations for curated auto-play and stations.
- **Lyrics View**: Full-screen lyrics mode with animated, blurred ambient backgrounds synced to track art.
- **Customizable Library**: Create playlists, favorite songs, and manage play history.
- **Dislike Algorithm**: Ban songs you don't like to refine your stations.
- **Premium UI**: Glassmorphism, tailored animations, and a seamless responsive design across desktop and mobile.

## Built With
- **React + Vite** for blazing fast UI
- **Zustand** for state management and local storage persistence
- **Framer Motion** for fluid micro-animations and layouts
- **Supabase** for database, auth, and real-time synchronization
- **Cloudflare Workers** for high-speed API brokering and proxying
